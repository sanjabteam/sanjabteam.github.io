# Cards

  • Create a card
  • Base properties
  • Availble cards

Cards are classes that showing any block in dashboard or crud.

# Create a card

Most simple card is a StatsCard.

use Sanjab\Cards\StatsCard;

$this->cards[] = StatsCard::create('Products')
    ->value(function () {
        return Product::count();
    });

# Base properties
