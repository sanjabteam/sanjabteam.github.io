# Number Widget

A simple input for numbers.

# Sample

use Sanjab\Widgets\NumberWidget;

$this->widgets[] = NumberWidget::create('age', 'Your age')
                            ->required()
                            ->min(18)
                            ->max(120);

You also should define $casts in your model.

protected $casts = [
    'age' => 'int'
];

# Properties

# min

type: number

minimum value.

# max

type: number

maximum value.

# step

type: number

the precision of each step of the number.
