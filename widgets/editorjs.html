# Editor.js widget

A WYSIWYG editor based on Editor.js.

# Sample

use Sanjab\Widgets\Wysiwyg\EditorJsWidget;

$this->widgets[] = EditorJsWidget::create('content');

You also should define $casts in your model.

protected $casts = [
    'content' => 'array'
];
